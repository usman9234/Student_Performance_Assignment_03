# Student Performance Analysis

This project analyzes the Students Performance in Exams dataset using Python.

## Dataset

Dataset Name: Students Performance in Exams  
CSV File: StudentsPerformance.csv  

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Git and GitHub

## Tasks Covered

1. Git and Version Control
2. Descriptive Statistics
3. Data Cleaning
4. EDA and Correlation
5. Data Visualization

## Purpose

The purpose of this project is to analyze student performance based on math, reading, and writing scores. The project includes descriptive statistics, cleaning, group analysis, correlation, and visualizations.
